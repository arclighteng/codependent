# Cursor Setup

AI-powered code editor (VS Code fork). **Codependent Sidecar.**

## Install

**macOS:** Download from https://cursor.com
**Windows:** Download from https://cursor.com

## Authentication

Cursor manages its own authentication via the app.

## Verify

```bash
command -v cursor
```

## Codependent Usage

Sidecar — not in the automatic failover chain. Always available
as a manual fallback. Launch with: `cursor .`

Guardrails are placed in `.cursor/rules/guardrails.mdc`
